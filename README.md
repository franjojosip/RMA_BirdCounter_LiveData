# Bird Counter with ViewModel Kotlin application

Basic Kotlin Application showing bird counter which is using local shared preferences and view model

In this application you can click on bird color you saw. It will trigger counter increment and change color of ImageView above.
Application use local shared preferences so when the application process is killed and application reopened it will show last known status. Data is loaded with ClickCounterViewModel and shown on screen.
